


Titlu : E-commerce Application

Proiectul reprezinta o mini aplicatie magazin-online de tip RESTfull, care implementeaza operatiunile CRUD. Securitatea se realizeaza prin autentificare cu JSON Web Token. Interfata utilizatorului este realizate cu ajutorul framework-ului Angular.

Tehnologii utilizate: Spring Boot;
				Spring Security;
				MySQL database;
				Angular framework;
				Limbaj de programare: Java
